# Mohammad Hosein Amini

## Profile Summary

I am a multidisciplinary software engineer focused on building scalable
systems, Android internals, and modern web applications.

My work spans Android (Kotlin, AOSP, system-level development),
full-stack web (Next.js, Node.js, Python), and DevOps automation.

I enjoy working close to the system layer --- customizing Android
builds, developing MDM solutions, automating Linux environments, and
optimizing deployment pipelines. At the same time, I design modern
front-end experiences and backend architectures built for performance
and simplicity.

I am currently expanding my expertise in Next.js, advanced Android
system customization, and scalable infrastructure design.

I am open to collaborating on technically challenging projects and
meaningful products.
